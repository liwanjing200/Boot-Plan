# 开机计划 · Boot Plan

先把自己开机，再开始改变人生。

Boot Plan 是从 `daily-negentropy-app` 独立复制出来的新版长期开发仓库。旧仓库保留原版；新版在保留原有数据逻辑的基础上，升级为深色、未来感、轻赛博的每日人生操作系统。

## 功能

- 每日任务与打卡、历史记录、每日复盘
- 财务中心 / 收支记录
- 妆容记录
- Journal 日记
- localStorage 本地保存逻辑，保留旧版 `negentropy_v3` 迁移
- 启动区、主任务区、轻任务区、恢复区、关机区
- 每天只能设置一个最重要任务
- 专注计时器、能量状态、连续开机天数、今日时间轴
- 数据中心：完成率、专注时长、趋势图
- 目标系统：英语、日语、妆容形象、AI/网站项目、赚钱与内容
- 深色 glassmorphism Dashboard，适配手机、iPad 和桌面端
- 六项今日复盘：完成、未完成、问题、明日改进、心情、备注
- 付款时间、项目、金额、支付方式和备注
- 按日期查看任务、复盘和付款历史
- 数据先保存在浏览器 `localStorage`，并同步到 Supabase 的 `boot-plan` 命名空间

## 使用

直接打开 `index.html` 即可运行。建议在 iPhone Safari 中添加到主屏幕使用。
